Tags com semântica em HTML5

Negrito, itálico e sublinhado

Para usar o negrito, usamos a tag <strong>
Para usarmos o itálico, usamos a tag <em>
Para usarmos o sublinhado, usamos a tag <ins>

Marca-texto, texto-deletado e abreviações

Para marcamos um texto, usamos a tag <mark>
Para deixarmos o texto como deletado, usamos a tag <del>
Para abreviarmos um texto como (HTML), podemos usar as tags <abbr title="">

Sobrescrito, subescrito e citações

Para deixarmos um número como X2 com um digito subescrito utilizamos a tag <sub>
Para deixarmos um número como X2 com um digito sobrescrito utilizamos a tag <sup>
Para citações simples como Machado de assis... usamos a tag <q>
